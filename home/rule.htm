标题：
家族风云榜规则
时间：
2006-03-10
 

1、  活动方式

1.1  玩家代表家族参加比赛，每个家族最多允许选派 50 名选手。家族参赛选手参加比赛获得积分，所有家族参赛选手积分之和为家族积分；

1.2  每周进行一个模式的比赛，总共包括：足球模式、爱的考验、比武模式三个个模式。

1.3  每日按照家族积分更新家族风云榜。每周根据一周比赛结果发布本周家族风云榜。所有模式比赛结束，公布最终的家族风云榜。

1.4  比赛分为电信和网通两个区域举行；

1.5  玩家可以购买鲜花赠送给参加比赛的选手以表示该选手所代表的家族的支持。当家族积分相同的情况下，根据家族参赛选手所拥有的鲜花总量多少决定名次；

3、  活动规则

3.1  比赛采用 1v1 模式；

3.2  比赛积分采用混战积分赛，玩家进入比赛频道，系统会自动进行随机配对，比赛局数不受限制

3.3  队伍和个人比赛的比赛成绩都将采用积分制度记录，积分规则如下：

　　胜利一场积 3 分；

　　平局一场积 1 分；

　　失败一场积 0 分。

3.4  在一个模式的比赛中，所有代表同一个家族的参赛选手积分之和为该家族的积分，根据家族积分排名 [ 家族风云榜 ] 。每个模式比赛开始积分清零。根据各模式比赛的家族积分之和排名最后的 [ 家族风云榜 ]

4、  参赛方法

4.1  创建家族

家族族长通过活动专区创建家族。创建流程如下：

第一步  验证登陆活动专区

第二步  创建家族，创建人条件： 100 个盟约书

第三步  填写家族相关资料

第四步  添加家族人员 qq 号（一个家族限 50 人）

比赛开始后，创建家族功能停止。无法添加新的家族成员。

4.2  参赛资格

家族族长最多只能添加 50 名比赛成员参赛，所有添加的家族成员都自动获得参赛资格。

4.3  察看比赛赛程

玩家登陆活动专区，可以察看自己比赛项目以及比赛时间。
