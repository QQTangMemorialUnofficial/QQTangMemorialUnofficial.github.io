《GCS互动社区》QQ堂核心用户成长计划

　　QQ 堂用心打造一款国产网络休闲游戏的品牌，从游戏开发到测试历经了几个层次的过度，终于在今夏推出了 “ 酷比一夏 ” 的正式版本。 QQ 堂的开发研制人员也在坚持不懈的为玩家提供更好的游戏质量和更宽松的游戏场景。 QQ 堂始终都重视那些时刻关注 QQ 堂成长历程的每位玩家。为了答谢 QQ 堂的忠实拥护者，下面推出 QQ 堂用户成长积分计划。

　　“ 用户成长计划 ” 是为了奖励那些一直以来致力于完善 QQ 堂而努力的玩家。我们将长期关注每位 QQ 堂玩家为 QQ 堂的成长所做的贡献，并且以完善的积分系统 ―― 积分累计系统 ―― 评定每一个 QQ 堂玩家的成绩，最终给予奖励。

　　( 一 ) 奖励机制：

1. 积分超过 10000 积分的普通玩家将可以一转成为 QQ 堂核心玩家。

2. 核心玩家中，积分达到 5000 0 以上，热心玩家联络工作，愿意帮助其他玩家解决问题，每周总在线时间不少于 20 小时的可申请二转成为 QQ 堂游戏巡查员。

3. 核心玩家中，积分达到 5000 0 以上，论坛表现活跃，写作能力强，多次发表 QQ 堂游戏相关作品的可申请二转成为 QQ 堂论坛版主和 QQ 堂记者团成员或 QQ 堂宣传团成员及 QQ 堂地图研究院成员。

4.QQ 堂游戏巡查员表现优秀者经选拔可以三转为终极职业 ―A 腾讯公司游戏 GM 客服。

5.QQ 堂论坛版主和记者团成员表现优秀者经选拔将可以三转为终极职业 ―B 腾讯公司游戏 CS 客服。

6.QQ 堂记者团成员表现优秀者经选拔也将可以三转为终极职业 ―C 腾讯公司签约游戏写手。

　　( 二 ) 积分系统：

1. 每 5000 “ 积分 ” 可以申请兑换 5000 0 糖币

2. 玩家提交的建议 /BUG 将分为四个等级，可获不同的积分。

S 级： 10000“ 积分 ”

A 级： 1000“ 积分 ”

B 级： 100“ 积分 ”

C 级： 10“ 积分 ”

　　( 三 ) 获得积分得方法：

　　1.Bug 提交的分级标准：

S 级： 重大游戏 bug ，如客户端死机崩溃，地图错误等；

A 级： 重要游戏 bug ，如对游戏造成很大影响的 Bug ，导致游戏无法进行下去等；

B 级： 普通游戏 bug ，如缺乏资源造成的 Bug 等；

C 级： 简单游戏 bug ，如显示错误，对话错误，界面错误等。

　　2. 建议提交的分级标准：

S 级：重大游戏建议，如报告外挂网站和制作团体，提出游戏建议被开发团队采纳，或提出反外挂建议被开发团队采纳等。

A 级：重要游戏建议，如有可行性的详细游戏改进方案或设计计划等。

B 级：普通游戏建议，如比较详细的游戏改进方案和计划等。

C 级：简单游戏建议，如言之成理但描述比较简单的游戏改进方案和计划等。

　　3. 玩家作品的分级标准：

S 级：玩家创作的水准较高的电影短片， flash ，游戏音乐，游戏小说，游戏漫画，手工制品等。

A 级：玩家写作的优秀技术性心得文章经产品组确认，玩家制作的一般水准的电影短片， flash 等。

B 级：游戏的论坛精华文章，玩家写作的一般技术性文章和优秀心情文章经版主或客服版主确认。

C 级：一般水平的心情文章和其他文章，经版主或客服版主确认。特别优秀的玩家作品除了获得积分外，将得到单独的稿酬。

　　( 四 ) 惩罚机制：

1. 使用、制造、传播恶性游戏外挂，利用和恶意传播游戏 bug ，扣除当前 “ 积分 ” 的 200% 。

2. 使用 QQ 堂游戏外挂者，第一次发现扣除当前 “ 积分 ” 的 10% ，第二次发现扣除当前积分的 20% ，以此类推，最低扣除数量为 1000“ 积分 ” 。

3. 在论坛被版主报警或申请禁言审核通过者，扣除 5000 “ 积分 ” 。

4. 在论坛被版主严重警告者，扣除 1000“ 积分 ” 。

5. 在论坛被版主警告者，扣除 100“ 积分 ” 。

6. 在游戏中被巡查员在游戏中禁言，严重警告和警告玩家，比照论坛相同的扣除原则处理。

7. 二转职业利用二转身份为自己或他人获取不正当利益，或恐吓，欺骗，侮辱游戏玩家的，经官方确认扣除当前 “ 积分 ” 的 50% 并立即免职。

8. 二转职业有其他失职行为的，经官方确认扣除当前 “ 积分 ” 分的 10% ，最低扣除数量为 5000 “ 积分 ” 。

9. 一转职业利用一转身份为自己或他人获取不正当利益，或恐吓，欺骗，侮辱游戏玩家的，经官方确认扣除当前 “ 积分 ” 的 30% ，最低扣除数量为 5000 “ 积分 ” 。

10. 一转职业在论坛和游戏中扣分按 120% 处理，二转职业按 200% 处理。

　　( 五 ) 核心玩家得权利：

1. 优先参加 QQ 堂新版本测试

2. 获得 QQ 堂开发运营团队提供的正式游戏资料和改版资料。

3. 优先参与 QQ 堂的各种线上线下活动和各种赠品。

4. 所有二转职业将具有核心玩家的全部权利，此外每月将获得官方发放的固定 Q 币工资。
