QQ堂里小鸟在歌唱--采访QQ堂内测明星╅葱葱"

　　QQ 堂里有英俊的火影，有俏皮的小倩，有美丽的玛利亚 …… 可是，就是没有小鸟啊，何来的鸟鸣呢？可是，在采访内测明星 `; ╅葱葱 ' 时我确实听到了小鸟在歌唱。

　　这个来自新疆的初三小女孩充满一种灵秀之气，宛如一只歌唱在 QQ 堂里的百灵鸟，尽情地书写自己的青春，挥霍着自己无穷的活力。刚一打照面，她就给了记者一个下马威：“等下回答你的问题好吗？我先问你几个问题？”原来小女孩还不知道为什么官方记者找上了他呢。她俏皮地说：“我……犯什么错误了？？”一脸的无辜！

　　╅葱葱 ' 原本也是记者团的，后来因为即将面临中考而放弃了，不过看得出来还挺怀念记者团那段日子的，一听说是官方记者就迫不及待地直问记者团现在的状况。从她的文章，根本看不出来是出自一名初三小孩的手笔，自不用说，小葱葱 ' 的作文是非常的了不起的，还经 常被 老师当范文在班上读出来呢。

　　╅葱葱 ' 从 QQ 堂第一次改版就开始玩了，算起来比记者的玩龄还要长呢。当问及玩 QQ 堂会不会惹来妈妈的怒火时，╅葱葱 ' 一脸无奈：“妈妈会骂我啊，谁让我脸皮这么厚，还是死心塌地得玩。”可玩归玩，╅葱葱 ' 还是在班上排名十一。

　　跟╅葱葱 ' 聊天是一件挺享受的事，这个可爱的小女孩不但文采洋溢，还会时不时幽上一默。“你滴采访怎么敢玩挖 ~ 彩不是不错，是相当 ~~~~~~~~~~ 不错！”这个可爱而体贴的小女孩呵，真让人疼到心里去了。

　　“对挖，高兴死我勒。。偶对着所有滴朋友吼呢！”这是╅葱葱 ' 对她得知赢得内测明星奖品时心情的描述。这个直率、整天乐呵呵的小女孩，无论是 QQ 堂还是现实，都拥有好多朋友，这是一件让人羡慕的事。

　　╅葱葱 ' 还傻气得很。在聊着聊着，她突然冒出一句“你什么时候开始采访啊？”记者一下子蒙了，原来小傻瓜还以为采访是要“先要问人要资料啊，事先要准备好一个档案的，然后要问问专业和关于一些喜好的问题，最后做个总结”呢。

　　这个冬天我听到了 QQ 堂里的第一声鸟鸣，心情是无比的欢欣与满足。最后，真诚地祝愿我们的内测明星╅葱葱 ' 永远那么快乐！
